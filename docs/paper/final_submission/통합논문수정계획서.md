## 2. 주요 수정 내용

### 2.1 긴급 수정 (Critical) ✅

- **Chow2025** "Hypothetical citation" → **Weidenfeller2022** (실제 IBM Quantum 논문)
- Fidelity 정의를 **Hellinger Fidelity**로 명확화

### 2.2 방법론 강화 ✅

- Eq.(2)에 $F_{crosstalk}$ 항 추가:

  ```
  F̂(C, n) = F_gate · F_decoherence · F_crosstalk
  ```

- $X_{ij}$ (Crosstalk intensity) 정의 및 데이터 소스(`zz_interaction`) 명시

### 2.3 실험 검증 확대 ✅

- **ZNE 정량화**: λ=1,3,5,7 스케일링 → Linear 외삽 20.9% 오차 감소
- **Scalability**: 5-15Q GHZ/QFT 시뮬레이션으로 하드웨어 검증 대체
- **Ablation**: Routing / Variant 기여도 분해
